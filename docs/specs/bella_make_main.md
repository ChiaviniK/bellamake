# Bella Make — main.md

## Visão
Sistema de gestão e PDV para loja de maquiagem com preço único de R$ 10,00. O foco é agilidade no atendimento e conformidade fiscal.

## Problema
Lojas de preço único possuem alto volume de transações pequenas. Sistemas genéricos são lentos e complexos demais para essa operação.

## Usuários principais
- **Operador de Caixa**: Realiza as vendas.
- **Gerente**: Gerencia estoque e fechamento financeiro.
- **Contador**: Acessa relatórios e arquivos fiscais.

## Métricas de sucesso
- Venda concluída em menos de 10 segundos.
- Disponibilidade de 99,9% para o PDV (suporte offline).
- Cobertura de testes de 80% no core financeiro/fiscal.

## Escopo inicial
- Frente de Caixa (PDV).
- Gestão de Estoque (Entrada via XML).
- Emissão de NFC-e.
- Relatórios de Fechamento.
