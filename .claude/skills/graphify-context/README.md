# Skill: graphify-context

Esta skill fornece integração com o sistema **Graphify**.

## Objetivo
Reduzir o consumo de tokens através de consultas semânticas ao grafo de conhecimento do projeto.

## Uso
Execute o comando `/graphify query "[pergunta ou contexto]"` para obter o subgrafo relevante.

## Saída
O resultado incluirá nós (arquivos/classes/métodos) e seus relacionamentos, facilitando o mapeamento de impacto sem a necessidade de ler todos os arquivos individualmente.
