# Skill: task-report

Esta skill automatiza a geração de relatórios de conclusão de tasks.

## Objetivo
Garantir que toda implementação seja documentada com métricas de economia de tokens e assertividade.

## Uso
Ao concluir uma task, o agente deve usar esta skill para preencher o template em `docs/tasks/YYYY-MM-DD-[feature].md`.

## Métricas
- Redução de tokens (comparativo com leitura naive).
- Assertividade do grafo (nós usados / nós surfaced).
